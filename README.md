# JavaScript To-Do List

A simple to-do list web app built with vanilla JavaScript.

## Overview
This project was created to practice JavaScript fundamentals by building an interactive web application. Users can add, delete, and manage tasks through a simple and intuitive interface.

## Features
- Add new to-do items
- Delete individual items with confirmation popup
- Scrollable task list to prevent overflow
- Responsive UI layout

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Git & GitHub

## What I Learned
- DOM manipulation to dynamically update UI
- Event handling for user interactions
- Basic state management using JavaScript arrays
- Connecting UI behavior with application logic
